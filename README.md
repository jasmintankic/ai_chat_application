# ai_chat_application
PraktiÄni dio predmeta "Umjetna inteligencija" (FIT 2. godina master studija) je chat aplikacija koja odgovara na postavljena pitanja.

0. Skinuti projekat
1. Instalirati Node.js - 4.3.1 LTS ( https://nodejs.org/en/ ) 
2. Otvoriti CMD unutar aiwebapp foldera
3. npm install -g grunt-cli bower
4. npm install
5. grunt serve
6. U browser ukucati http://localhost:9000/index.html
